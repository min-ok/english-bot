# Лендинг «Слова в контексте»

Статический одностраничный лендинг для Telegram-бота
[@spaced_repetition_in_context_bot](https://t.me/spaced_repetition_in_context_bot).
Чистые HTML/CSS/JS, без сборки — на Netlify заезжает как есть.

## Файлы

- `index.html` — вся разметка страницы.
- `styles.css` — стили (адаптивные, mobile-first).
- `script.js` — открытие/закрытие мобильного меню.

## Что нужно вписать вручную

- Раздел «Отзывы» (`id="reviews"` в `index.html`) — сейчас там 6 заготовок
  с плейсхолдерами «Впишите сюда текст отзыва» / «Имя Фамилия». Замените
  текст, имя и (если нужно) букву в кружке-аватарке на реальные.

## Деплой на Netlify через GitHub

1. Создайте новый репозиторий на GitHub и запушьте в него содержимое этой папки.
2. В Netlify: **Add new site → Import an existing project → GitHub** → выберите репозиторий.
3. Build command — оставить пустым, Publish directory — `.` (корень репозитория).
4. Deploy — сайт будет доступен на выданном Netlify-домене, при желании подключите свой.
